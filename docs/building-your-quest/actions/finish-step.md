# finish_step

## Description

**Type**: Action

Advances the user to the next step in the current quest, or finishes the quest if this is the last step.

## Params

No params required.

## Outputs

No additional info is added to the global payload outputs.

## Usage Example

```yaml
do:
- actionId: finish_step
```
